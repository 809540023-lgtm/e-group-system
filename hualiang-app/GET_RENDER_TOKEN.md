# 📋 獲取 Render API Token 指南

為了自動部署，我需要你的 **Render API Token**。

## 🔑 獲取 API Token (2 分鐘)

### 步驟 1️⃣ - 登入 Render

1. 訪問 https://dashboard.render.com
2. 使用 cia8885@gmail.com 登入

### 步驟 2️⃣ - 進入 API 設置

1. 點擊右上角 **頭像** → **Account**
2. 左側菜單選擇 **"API Tokens"**
3. 點擊 **"Create API Token"**

### 步驟 3️⃣ - 複製 Token

1. 給 Token 一個名稱，例如：`hualiang-deployment`
2. 點擊 **"Create Token"**
3. **立即複製** Token (只顯示一次！)

### 示例 Token 格式：
```
rnd_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
