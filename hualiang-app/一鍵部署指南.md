### 第 3 步：部署後端 API (2 分鐘)

在 **Create a New Web Service** 頁面：

**Basic Settings:**
```
Name: hualiang-api
Repository: 809540023-lgtm/e-group-system
Branch: main
Root Directory: hualiang-app/backend
Runtime: Python 3
```

**Build & Deploy:**
```
Build Command: pip install -r requirements.txt
Start Command: gunicorn -w 4 -b 0.0.0.0:$PORT app:create_app()
```

**Environment Variables:**
點擊 **"Add Environment Variable"** 添加：
```
FLASK_ENV = production
FLASK_DEBUG = False
SECRET_KEY = hualiang-2025-secret-production
```

**點擊藍色按鈕：【Create Web Service】**

⏳ 等待 3-5 分鐘...

✅ 後端已部署！複製你的 API URL：
```
https://hualiang-api.onrender.com
```

## 🐛 遇到問題？

### 前端顯示空白？
- 檢查環境變量 `VITE_API_URL` 是否正確
- 確認後端已部署並運行

### API 無法連接？
- 訪問 Render Dashboard
- 找到 hualiang-api 服務
- 查看 "Logs" 標籤找出問題

### 部署失敗？
- 在 Render Dashboard 查看 "Logs"
- 通常是依賴或配置問題
- 點擊 "Redeploy" 重試
