====================================
🚀 快速啟動命令
====================================

# 後端 (終端 1)
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

# 前端 (終端 2)
cd frontend
npm install
npm run dev

# 管理後台 (終端 3)
cd admin
npm install
npm run dev

🎯 核心功能
====================================

用戶側:
✅ 會員目錄 (按行業/地區篩選)
✅ 商機匹配 (紅卡需求 + 綠卡優惠)
✅ 活動管理 (即將/往期活動)
✅ 生活分享 (會員動態相片)
✅ 消息公告 (最新資訊)

管理側:
✅ 儀表板 (實時統計)
✅ 會員管理 (CRUD)
✅ 活動管理 (CRUD)
✅ 消息管理 (CRUD)
✅ 商機管理 (CRUD)

🛠️ 技術棧
====================================
前端: React 18 + Vite + Tailwind CSS + Lucide Icons
後端: Flask 2.3 + SQLAlchemy + SQLite
API: RESTful + Flask-CORS
